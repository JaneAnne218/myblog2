axios概念 ​

axios是一个基于Promise的HTTP客户端，用于浏览器和node.js。axios本身具有以下特征：

  • 从浏览器中创建XMLHttpRequests
  • 从node.js创建http请求
  • 支持Promise API
  • 拦截请求和响应
  • 转换请求数据和响应数据
  • 取消请求
  • 自动转换JSON数据
  • 客户端支持防御XSRF

1. 首先，在命令行中创建一个基本的Node.js项目，并安装axios库。 ​

shell
npm init -y 
npm install axios

2. 在项目根目录下创建一个index.js文件，并在其中编写代码。该代码会向一个API服务器发送GET请求，并输出响应数据。 ​

js
const axios = require('axios');

axios.get('https://jsonplaceholder.typicode.com/posts/1')
  .then(response => {
    console.log(response.data);
  })
  .catch(error => {
    console.error(error);
  });

3. 运行该文件，查看控制台输出结果。 ​

shell
node index.js

以上代码演示了如何使用axios库发起HTTP请求并处理响应数据

axios.create() axiosInstance.get() ​

js
this.axiosInstance=axios.create()
this.axiosInstance.get()

引入axios ​

common js的方法引入axios:

js
const axios = require('axios');
console.log(axios)
